# AP Process Navigator

AP Process Navigator is a production-ready static website that converts an Accounts Payable workflow map into an interactive decision-support tool. It helps users navigate email intake, Zone routing, PO validation, IR review, utility invoice handling, and exception escalation.

## Features

- Fully static HTML, CSS, and vanilla JavaScript
- Responsive dark glassmorphism interface with purple accents
- Combined AP workflow decision tree with Email and Zone Tracker start options
- Data-driven workflow objects for easy future edits
- Yes/No decision buttons, back, restart, progress tracker, and breadcrumb path
- Learning Mode and Quick Decision Mode
- Copy next step and copyable exception email templates
- Searchable Exception Guide
- Tools Glossary for AP systems, roles, and documents
- Mock KPI Dashboard with cards and CSS/JavaScript charts

## How to run locally

Open `index.html` directly in a browser.

You can also use any simple static file server, but no server or build step is required for the site to work.

## How to deploy to GitHub Pages

1. Create a GitHub repository.
2. Add `index.html`, `styles.css`, `script.js`, and `README.md` to the repository root.
3. Commit and push the files.
4. In GitHub, open **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the branch, usually `main`, and the root folder.
7. Save. GitHub Pages will publish the static site.
